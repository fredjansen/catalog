Statistical Features

Home > Statistical Features

In this topic, we clustered all statistical features that characterize sites selected to this research. The features are:

Daily audience

This feature indicates the daily average of people that access the site. The value expected is a number.

Total users registered

This feature indicates total users registered in the site. The response is a number.

Question rated

This feature indicates how the questions are rated. Expected values: Upvoted by users, Upvoted by community, Liked by users, Liked (reactions), Editorial selection, Reactions. When there are no alternatives, put None. Allow multiple values.

Answer rated

This feature indicates how the answers are rated. Expected values: Upvoted by users, Upvoted by community, Liked by users, Liked (reactions), Editorial selection, Reactions. When there are no alternatives, put None. Allow multiple values.

Percentual answered

This feature indicates the percentage of questions answered on the site. The response is a number between 0 and 100.

Average number of interactions per question

This feature indicates the average number of interactions in each question on the site. The response is a number.

Total of citations

This feature indicates the total of citations of the site in journals, papers and other scientific documents. The response is a number.

Questions for Statistical Features

  1. Based on the "Daily audience" explanation, What is the size of this site's daily audience? The expected value is a number.
  2. Based on the "Total users registered" explanation, How many users are registered on this site? The expected value is a number.
  3. Based on the "Question rated" explanation, How is the question rated on the site? The expected values: Upvoted by users, Upvoted by community, Liked by users, Liked (reactions), Editorial selection, Reactions. When there are no alternatives, put None. Allow multiple values.
  4. Based on the "Answer rated" explanation, How is the answer rated on the site? The expected values: Upvoted by users, Upvoted by community, Liked by users, Liked (reactions), Editorial selection, Reactions. When there are no alternatives, put None. Allow multiple values.
  5. Based on the "Percentual answered" explanation, What percentage of the questions on this site are answered? The expected value is a number between 0 and 100.
  6. Based on the "Average number of interactions per question" explanation, How many interactions per question on this site? The expected value is a number.
  7. Based on the "Total of citations" explanation, How many external citations such as journals, articles does this website have? The expected value is a number.
