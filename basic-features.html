In this topic, we clustered all basic features that characterize sites selected to this research. The features are:

Associated Site

This feature indicates which platform this site belongs to. Expected values: reddit, stack exchange, quora and others. Based on this explanation above, answer the question. What platform is this site linked to? The expected values: reddit, stack exchange, microsoft, quora. When there are no alternatives, put None.

Creation Date

This feature indicates what year the site was created. The value expected is a number that represents a year that site was created. Based on this explanation above, answer the question. What is the creation date? The expected value is a number that represents the creation year.

Website Organization

This feature indicates the way the site is organized. Expected values: in topics, in areas, in communities and other ways. Based on this explanation above, answer the question. How are these sites organized? The expected values: in topics, in areas, in communities. When there are no alternatives, put None.

Registration Methods

This feature indicates how users log in to navigate in the site. Expected values: by own registration, by facebook account, by apple id, by gmail account, by linkedin, by github account, by outlook account and other ways of registration. Based on this explanation above, answer the question. What are the available registration methods? The expected values: own registration, apple ID, gmail account, linkedin, github account, outlook account. Allow multiple values.

Search Engines

This feature indicates ways of search engines inside the site. Expected values: string search, tags/labels search, similarity search and other ways. Based on this explanation above, answer the question. What search engines are available on this site? The expected values: string search, tags/labels search, similarity search. When there are no alternatives, put None. Allow multiple values.

Type of Audience

This feature indicates the target technical profile of the website. Expected values: Developers, Software Engineers, Software Architects, Testers or others. Based on this explanation above, answer the question. What type of audience does this site target? The expected values: developers, software engineers, software Architects, testers. When there are no alternatives, put IT professionals. Allow multiple values.

Website Type

This feature indicates the type of website. Expected values: site, forum, community, blog. Based on this explanation above, answer the question. Is this site a forum, a community, a blog or something else? The expected values: forum, community, blog or website.

Subarea of Software Engineering

This feature indicates which sub-area of software engineering the site is most focused on. Expected values: general, coding, architecture, maintenance, testing, management, and evolution. Allow multiple values. Based on this explanation above, answer the question. Which of the sub-areas within software engineering? The expected values: general, coding, architecture, maintenance, testing, management, and evolution. Allow multiple values.
