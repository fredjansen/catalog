Home > Connection and Data Features

In this topic, we clustered all connection and data features that characterize sites selected to this research. The features are:

Has its own API

This feature indicates whether the site has its own API or not. Expected values: yes or no.

Can download repository

This feature indicates whether the site can download or not repository. Expected values: can or cannot.

Has its own Dashboard

This feature indicates whether the site has its own dashboard or not. Expected values: yes or no.

Has its app

This feature indicates whether the site has its own app or not. Expected values: yes or no.

Questions for Connection and Data Features

  1. Based on the "Has its own API" explanation, Does this site have its own API for searching data? The expected values: yes or no.
  2. Based on the "Can download repository" explanation, Does this site allow you to download a repository? The expected values: can or cannot.
  3. Based on the "Has its own Dashboard" explanation, Does this site have its own dashboard with general information? The expected values: yes or no.
  4. Based on the "Has its app" explanation, Does this site have its own apps? The expected values: yes or no.
